**How future users change the model:**
1. Interactive: run `claude` as `wayan` → `/model <name>` (saves as default), or
2. Edit `~wayan/.claude/settings.json` and set `"model": "<id>"`.
3. The gateway passes no `--model`, so agents always inherit this default — no
   code change or service restart needed (each message spawns a fresh `claude -p`).
4. **Never hardcode a model that isn't verified**: test first with
   `claude -p --model <id> "Reply OK"`.
