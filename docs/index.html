Section 1

What you will build

Two personal AI agents on your own server, reachable from Telegram — with long-term memory and a versioned knowledge base.

Jupiter

The primary research / content / analysis agent: tokens, PDFs, voice notes, videos, posts.

Uran

The operations / server / support agent: logs, diagnostics, restarting Jupiter when it falls over.

OpenViking memory

Semantic long-term memory: the agents recall knowledge by meaning, localhost-only.

Visual video analysis

Groq audio transcript + ffmpeg keyframes + Claude visual analysis — one merged review.

WAYAN_PACK presets

Profession presets at install time: onchain, creator, devops, student, founder.

Repo-first deployment

GitHub as the source of truth: pull → tests → drift check → backup → deploy.

Section 2

Choose your path

Beginner — read the guide first. Open the Russian or English guide: it teaches every concept before asking you to run anything.
Builder — install on a VPS. Follow the 18-step install flow in the guide: VPS → SSH → installer → bots → Claude login → memory → tests.
Developer — fork and customize. Fork the repository, rewrite templates, skills, and packs — secrets always stay local.
Creator / analyst / operator — choose a profession pack. WAYAN_PACK=creator sudo -E ./install.sh — the same system, tuned for your work from day one.
