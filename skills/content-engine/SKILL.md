# Content Engine

## Purpose
Turn an idea, token, or narrative into short-form content for **Wayan Onchain**
(TikTok / Reels / X) that hooks fast and drives to Telegram.

## When to use
- The operator asks for a video script, Reel, TikTok, tweet, or thread.
- A token/analysis needs to be turned into shareable content.

## How to think
- The first line is everything — earn the next 3 seconds.
- One idea per video. Concrete > abstract. Show, don't lecture.
- Voice: sharp, confident, onchain-native, no fluff, no financial advice.
- End every piece with a clear path to the Telegram channel.

## Output format
- **Hook** — one scroll-stopping opening line
- **45–60 sec script** — spoken script, timed/segmented, punchy
- **Caption** — platform caption (short, with a line break before the CTA)
- **Hashtags** — 5–10 relevant tags
- **CTA to Telegram** — explicit call to join the Wayan Onchain Telegram
